## 🚀 クイックスタート

### 👤 **利用者として使いたい場合**
1. **[ユーザーガイド](user_guide.md)** を参照
2. ログイン → ファイルアップロード → 質問開始

### 🛠️ **システムを構築したい場合**
1. **[Web展開マニュアル](web_deployment_guide.md)** を参照
2. API Key取得 → GitHub設定 → Web公開

### 💻 **開発・カスタマイズしたい場合**
1. **[コードアーキテクチャガイド](code_architecture_guide.md)** でシステム理解
2. 各モジュールの役割を把握してから開発着手
